# Step 1: 去遮挡角色头部四视图

## 目标

生成角色的去遮挡头部四视图设定图。

输出：

- 正面
- 左前45度
- 左侧面
- 背面

## 上传图片

建议上传 4-7 张：

1. 角色正面结构参考图
2. 角色背面结构参考图
3. 角色侧面或45度结构参考图
4. 角色目标表情参考图
5. 官方2D图，用于气质和角色识别
6. 如果角色有遮挡，上传无遮挡3D模型图或游戏内截图

不要上传：

- Kigurumi头壳案例图
- 商品照案例图
- 其他角色图
- 过多重复图

## 通用 Prompt

```text
请根据我上传的参考图，生成这个角色“去遮挡后的头部四视图设定图”。

参考图分工如下：
1. 结构参考图：用于确定角色的头部结构、发型轮廓、耳朵位置、侧脸轮廓、后脑和背面发型。
2. 表情参考图：用于确定角色的眉毛、眼神、眼睑形状、嘴巴和整体表情。
3. 官方角色图：用于确定角色气质、配色、虹膜颜色和角色识别感。

请生成一张干净的角色头部四视图设定图，包括：
正面、左前45度、左侧面、背面。

要求：
- 去掉遮挡物，例如兜帽、帽子、面罩等，但保留角色必要的小发饰。
- 只保留头部、头发、耳朵和必要饰品。
- 保留角色原本的表情、眼神和气质。
- 发型结构应根据结构参考图合理补全。
- 不要改成普通萌系脸。
- 不要把脸做得太3D、太写实、太立体。
- 不要生成kigurumi头壳。
- 不要生成真人、cosplay、娃娃、手办、3D渲染风格。
- 白色背景，角色设定图风格，结构清晰。
- 四个视角必须是同一个角色、同一个发型、同一个表情系统。
```

## 表情修正 Prompt

```text
请基于当前四视图设定图进行微调，不要重新设计角色。

当前发型结构和四视图一致性基本正确，请保持不变。
这次只重点修正表情。

请根据目标表情参考图调整：
- 眉毛角度
- 眼神
- 上眼睑和下眼睑形状
- 嘴巴形状
- 整体情绪

要求：
- 表情更接近参考图。
- 不要自动微笑，除非参考图本身就是微笑。
- 不要改成普通可爱表情。
- 不要让表情过度夸张。
- 保持角色原本气质。

输出仍然是同一个角色的头部四视图设定图。
```

## 正脸修正 Prompt

```text
请基于当前四视图设定图进行微调，不要重新设计角色。

当前发型和四视图一致性基本正确，请保持不变。
这次重点修正正脸五官。

请根据正脸参考图调整：
- 眼睛形状
- 眼距
- 眉眼关系
- 嘴巴位置
- 脸部比例
- 虹膜颜色和眼神

要求：
- 正脸更接近目标角色。
- 不要改变发型结构。
- 不要改变角色身份。
- 不要变成普通模板脸。
- 不要把脸变得更3D、更写实或更立体。

输出仍然是角色头部四视图设定图。
```

## 脸型幼态化 Prompt

```text
请基于当前四视图设定图进行脸型修正，不要重新设计角色。

当前问题是脸型偏尖、偏成熟，不够幼态。

请重点调整：
- 下巴缩短。
- 下巴更圆、更钝、更小。
- 下半张脸缩短。
- 脸颊更饱满。
- 下颌线更柔和。
- 整体脸型更圆润、更幼态。
- 不要长脸。
- 不要尖下巴。
- 不要成熟脸。

同时保持：
- 发型结构不变。
- 表情不变。
- 角色识别度不变。
- 四视图排版不变。

输出仍然是同一个角色的头部四视图设定图。
```
