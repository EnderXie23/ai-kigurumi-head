# Troubleshooting

## 如何向 ChatGPT 求助改 Prompt

当生成结果不理想时，提供以下信息：

1. 当前执行的是哪一步：Step 1 / Step 2 / Step 3
2. 上传了哪些参考图，以及每张图的作用
3. 当前使用的 Prompt
4. 生成结果哪里正确
5. 生成结果哪里错误
6. 希望保留什么
7. 希望修改什么

## 求助模板

```text
我现在在执行 Step [数字]。

我上传的参考图包括：
1. [图片A作用]
2. [图片B作用]
3. [图片C作用]

我使用的 Prompt 是：
[粘贴当前 Prompt]

生成结果的问题是：
1. [具体问题1]
2. [具体问题2]
3. [具体问题3]

我希望保留：
- [已经正确的部分]

我希望修改：
- [需要修改的部分]

请帮我分析问题原因，并给出一版更强、更明确的修正 Prompt。
```

## 推荐的问题描述方式

不推荐：

```text
不好看。
不像。
怪。
```

推荐：

```text
脸型太尖，下半张脸太长，角色显得太成熟。
```

```text
表情不对，参考图是冷淡无表情，但生成结果变成了微笑。
```

```text
头壳感不足，看起来像普通动漫头像，没有眼眶结构和硬质面壳感。
```

```text
太像3D娃娃，鼻子和嘴唇太立体，有明显唇彩。
```

## 常见问题分类

### 表情不对

检查：

- 是否上传了明确的目标表情图
- Prompt 是否写清楚眉毛、眼神、眼睑、嘴巴
- 是否要求不要自动微笑

### 角色不像

检查：

- 是否让角色设定图作为主参考
- 是否上传了太多风格不一致的图片
- 是否让 Kigurumi 案例污染了角色五官

### 脸型太尖

使用具体几何描述：

- 下巴缩短
- 下半脸缩短
- 下巴更圆更钝
- 脸颊更饱满
- 下颌线更柔和

### 头壳感不足

加强：

- 硬质平滑面壳
- 固定表情
- 眼眶结构
- 假发作为真实假发存在
- 侧面头壳厚度

### 太 3D / 太娃娃

削弱：

- 鼻子立体度
- 嘴唇和唇彩
- 真实皮肤纹理
- BJD 娃娃感
- 3D 渲染感

### 多视图不一致

强调：

- 同一个角色
- 同一个头壳
- 连续旋转展示
- 发型、耳朵、脸型、表情保持一致
