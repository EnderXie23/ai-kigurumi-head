# Additional Terms / 附加条款

This project is intended to help individual Kigurumi players create better character reference sheets and improve communication quality with makers.

本项目旨在帮助 Kigurumi 玩家制作更好的角色参考图，并提升与头壳店家的沟通质量。

---

## Allowed Uses / 允许的使用方式

You may:

- Use this workflow for personal reference generation.
- Send generated reference sheets to makers for communication.
- Modify prompts for personal or non-commercial use.
- Share results within the community with attribution.

你可以：

- 将本工作流用于个人参考图生成。
- 将生成结果发送给店家作为制作沟通参考。
- 为个人或非商业用途修改 Prompt。
- 在社区内署名分享生成结果。

---

## Restricted Commercial Uses / 受限制的商业使用

Without explicit permission from the project author, commercial entities or makers may NOT:

- Charge additional fees specifically for using this workflow.
- Market this workflow as a paid value-added service.
- Repackage the prompts/workflow into commercial products or closed services.
- Sell modified versions of the workflow or prompts.
- Use this project as a justification for increasing head shell pricing.

未经项目作者明确许可，商业实体或头壳店家不得：

- 以“使用本工作流”为理由额外收费。
- 将本工作流包装为付费增值服务。
- 将 Prompt 或工作流重新打包为商业产品或闭源服务。
- 售卖修改版 Prompt 或工作流。
- 以本项目为理由提高头壳制作收费。

---

## Community Spirit / 社区原则

This project is designed to empower players, not to increase commercialization pressure within the community.

本项目的目标是帮助玩家，而不是增加社区商业化压力。

Please respect the spirit of fair use and community collaboration.

请尊重公平使用与社区协作精神。
